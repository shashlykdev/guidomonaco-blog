# Guido Monaco Florence Blog

SEO-optimized blog for Guido Monaco Florence apartment.

## Features

- Next.js 14 (App Router)
- Static site generation
- Markdown blog posts
- Tailwind CSS styling
- SEO optimized
- Deployed on Vercel

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy

Deployed automatically via Vercel when pushing to GitHub.

## Content

Add new blog posts in `/content/` as Markdown files with frontmatter:

```markdown
---
title: "Your Post Title"
description: "Meta description"
date: "2026-02-08"
author: "Guido Monaco Team"
keywords: "keyword1, keyword2"
---

# Your Content Here
```
